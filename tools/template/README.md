# ToolPageLayout Template

## Overview
This is the reusable HTML template for generating SEO-optimized tool landing pages.
Each page is generated by replacing `{{VARIABLE}}` placeholders with actual content.

## Template Variables

### Meta & SEO
| Variable | Description | Example |
|----------|-------------|---------|
| `{{LANG}}` | Language code | `en` or `ar` |
| `{{DIR}}` | Text direction | `ltr` or `rtl` |
| `{{TITLE}}` | Page title (50-60 chars) | `Merge PDF Files Free Online - Stirling PDF` |
| `{{META_DESCRIPTION}}` | Meta description (150-160 chars) | `Merge multiple PDF files...` |
| `{{META_KEYWORDS}}` | Comma-separated keywords | `merge pdf, combine pdf...` |
| `{{SLUG}}` | URL slug | `merge-pdf` |
| `{{OG_TITLE}}` | Open Graph title | Same as TITLE |
| `{{OG_DESCRIPTION}}` | Open Graph description | Same as META_DESCRIPTION |
| `{{OG_LOCALE}}` | OG locale | `en_US` or `ar_AR` |

### Content Sections
| Variable | Description |
|----------|-------------|
| `{{H1}}` | Main H1 heading with primary keyword |
| `{{HERO_SUBTITLE}}` | Subtitle under H1 (value proposition) |
| `{{CONTENT_WHAT_IS}}` | "What is" section HTML content |
| `{{HOWTO_STEPS_HTML}}` | How-to steps rendered as HTML |
| `{{FEATURES_HTML}}` | Feature cards grid HTML |
| `{{USECASES_HTML}}` | Use case cards HTML |
| `{{CONTENT_TIPS}}` | Tips & best practices HTML |
| `{{FAQ_HTML}}` | FAQ accordion items HTML |
| `{{RELATED_TOOLS_HTML}}` | Related tools cards HTML |

### Schema Data
| Variable | Description |
|----------|-------------|
| `{{HOWTO_TITLE}}` | HowTo schema name |
| `{{HOWTO_DESCRIPTION}}` | HowTo schema description |
| `{{HOWTO_STEPS}}` | HowTo schema steps JSON |
| `{{FAQ_SCHEMA_ITEMS}}` | FAQPage schema items JSON |
| `{{TOOL_NAME}}` | Tool name for SoftwareApplication schema |

### Navigation & UI
| Variable | Description |
|----------|-------------|
| `{{NAV_HOME}}` | Home link text ("Home" / "الرئيسية") |
| `{{CTA_PRIMARY}}` | Primary CTA button text |
| `{{CTA_OPEN_TOOL}}` | Open tool button text |
| `{{CTA_FINAL_TITLE}}` | Final CTA section heading |
| `{{CTA_FINAL_SUBTITLE}}` | Final CTA description |
| `{{CTA_FINAL_BUTTON}}` | Final CTA button text |
| `{{BREADCRUMB_HOME}}` | Breadcrumb home text |
| `{{BREADCRUMB_TOOLS}}` | Breadcrumb tools text |
| `{{BREADCRUMB_CURRENT}}` | Breadcrumb current page |
| `{{LANG_EN_ACTIVE}}` | Active class for EN link |
| `{{LANG_AR_ACTIVE}}` | Active class for AR link |

### Section Titles
| Variable | Description |
|----------|-------------|
| `{{SECTION_TOOL_TITLE}}` | Tool interface section heading |
| `{{SECTION_WHAT_IS_TITLE}}` | "What is" section heading |
| `{{SECTION_HOWTO_TITLE}}` | "How to" section heading |
| `{{SECTION_FEATURES_TITLE}}` | Features section heading |
| `{{SECTION_USECASES_TITLE}}` | Use cases section heading |
| `{{SECTION_TIPS_TITLE}}` | Tips section heading |
| `{{SECTION_FAQ_TITLE}}` | FAQ section heading |
| `{{SECTION_RELATED_TITLE}}` | Related tools section heading |

## Schemas Included
1. **BreadcrumbList** — Navigation path
2. **SoftwareApplication** — Tool metadata (free, web-based)
3. **HowTo** — Step-by-step instructions
4. **FAQPage** — Questions and answers

## Usage
```python
# In generation script:
template = open('tools/template/tool-page-layout.html').read()
page = template.replace('{{SLUG}}', 'merge-pdf')
                .replace('{{TITLE}}', 'Merge PDF Files Free...')
                ...
```
