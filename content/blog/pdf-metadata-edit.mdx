# How to Edit PDF Metadata (Title, Author, Date)

> **Short answer.** To edit PDF metadata, open [Update Metadata](/tools/update-metadata), change the title, author, or other fields, upload the file, and download the updated copy.

PDF metadata is invisible to readers but visible to search engines, file managers, and document management systems. A file titled "final_v3_FINAL.pdf" in the metadata looks unprofessional. Fix it in seconds.

## What you'll need

- A PDF with metadata you want to change.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Check current metadata.** Right-click the file, check properties (Windows) or Get Info (Mac). Note the Title, Author, Subject fields.
2. **Open the tool.** Go to [Update Metadata](/tools/update-metadata).
3. **Enter new values.** Set the title, author, and any other fields you want to change.
4. **Upload.** Drop the PDF.
5. **Download.** The metadata is updated; content unchanged.

## Why metadata matters

- **SEO for shared PDFs.** Search engines index PDF metadata. A good title helps discoverability.
- **Professional appearance.** Document management systems display the metadata title, not the filename.
- **Privacy.** The author field may reveal who created the document. Clear it before sharing externally.
- **Compliance.** Some regulatory submissions require specific metadata fields.

## Common pitfalls

- **Forgot to clear the author.** Before sharing externally, remove personal names if privacy matters.
- **Metadata survives compression.** [Compress PDF](/tools/compress-pdf) doesn't strip metadata. Edit it explicitly.
- **Creation date reveals timing.** If you need to anonymize when a document was created, edit the date fields.

## Frequently asked questions

### Is it free to edit PDF metadata?

Yes. [Update Metadata](/tools/update-metadata) is free, no signup.

### Does editing metadata change the content?

No. Only the invisible properties change. The pages, text, and images remain identical.

### What fields can I edit?

Title, Author, Subject, Keywords, Creator, Producer, Creation Date, Modification Date.

### Is my file safe?

Public tools see your file. Self-hosted Stirling-PDF keeps it local.

### Can I remove all metadata?

Set all fields to empty. The tool will write blank values, effectively removing the information.

## Wrap-up

PDF metadata is often overlooked but matters for professionalism, privacy, and SEO. Open [Update Metadata](/tools/update-metadata), set clean values, download.
