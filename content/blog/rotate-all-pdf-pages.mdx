# How to Rotate All Pages in a PDF at Once

> **Short answer.** To rotate all pages in a PDF at once, open [Rotate PDF](/tools/rotate-pdf), pick the angle (90°, 180°, or 270°), upload the file, and download the corrected version. Every page rotates together.

A scanner saved your 20-page document sideways. Or a colleague exported landscape slides as portrait. You need every page turned the same direction — not one at a time, all at once.

## What you'll need

- A PDF with pages in the wrong orientation.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Rotate PDF](/tools/rotate-pdf).
2. **Pick the angle.** 90° clockwise is the most common fix for sideways scans. 180° for upside-down. 270° for counter-clockwise.
3. **Upload.** Drop the PDF.
4. **Rotate.** Click the action button.
5. **Download.** All pages are now rotated.

If only some pages are sideways and others are correct, you'll need to [Extract Pages](/tools/extract-pages) the sideways ones, rotate them, then [Merge PDFs](/tools/merge-pdf) everything back in order.

## When to rotate vs. when to re-scan

- **Rotate** when the content is correct but the orientation is wrong (common with flatbed scanners).
- **Re-scan** when the content is crooked at an angle (not exactly 90° off). Rotation only does 90° increments.

## Common pitfalls

- **Text is still sideways after rotation.** You probably picked 180° when you needed 90°. Try again with the other angle.
- **Mixed orientations.** Some pages landscape, some portrait. Split, rotate the wrong ones, merge back.
- **File gets larger.** Rotation itself doesn't change file size. If the output is larger, it's a metadata expansion — run [Compress PDF](/tools/compress-pdf).

## Frequently asked questions

### Is it free to rotate a PDF?

Yes. [Rotate PDF](/tools/rotate-pdf) is free, no signup, no watermark, no page limit when self-hosted.

### Can I rotate just one page?

Not with this tool directly. Workaround: [Extract Pages](/tools/extract-pages) the one page, rotate it, [Merge PDFs](/tools/merge-pdf) it back.

### Does rotation affect text searchability?

No. The text layer rotates with the page. Search, copy, and OCR still work.

### Is my file safe?

Public tools see your file. Self-hosted Stirling-PDF keeps it local. For sensitive documents, self-host.

### Can I rotate and compress at the same time?

Run them sequentially: rotate first with [Rotate PDF](/tools/rotate-pdf), then [Compress PDF](/tools/compress-pdf). Order doesn't matter for the result.

## Wrap-up

Rotating all pages in a PDF at once is a single-click operation. Open [Rotate PDF](/tools/rotate-pdf), pick your angle, upload, download. For mixed-orientation documents, combine with [Extract Pages](/tools/extract-pages) and [Merge PDFs](/tools/merge-pdf).
