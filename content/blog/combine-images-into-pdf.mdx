# How to Combine Multiple Images into One PDF

> **Short answer.** To combine multiple images into one PDF, open [Image to PDF](/tools/jpg-to-pdf), drop all your images in, reorder them by dragging, and click convert. One PDF comes out with each image on its own page.

You photographed a 10-page document with your phone and now have 10 separate JPGs. Or you exported slides as PNGs and need them in a single file for email. Same solution: images in, one PDF out.

## What you'll need

- Two or more image files (JPG, PNG, GIF, TIFF, WebP).
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Image to PDF](/tools/jpg-to-pdf).
2. **Add images.** Drop them all at once or click to select. Order matters — the tool respects your upload order.
3. **Reorder.** Drag thumbnails if the sequence is wrong.
4. **Convert.** Click the action button.
5. **Download.** One PDF, one page per image.

If the images are phone photos of a document and you want the text searchable afterward, follow up with [OCR PDF](/tools/ocr-pdf).

## Why combine images into a PDF

- **Email attachments.** One file is easier to attach and track than 10.
- **Print queue.** Printers handle multi-page PDFs better than a folder of images.
- **Archive.** A PDF with embedded images is self-contained; it won't break if you move it.

## Common pitfalls

- **Images in the wrong order.** Always reorder thumbnails before converting. The tool uses display order, not filename order.
- **Huge output file.** Phone photos are often 3–5 MB each. After combining, run [Compress PDF](/tools/compress-pdf) to shrink the result.
- **Sideways photos.** If a photo was taken in portrait but saved as landscape, use [Rotate PDF](/tools/rotate-pdf) on the output to fix individual pages.

## Frequently asked questions

### Is it free to combine images into a PDF?

Yes. [Image to PDF](/tools/jpg-to-pdf) is free, no signup, no watermark, no page limit when self-hosted.

### What image formats are supported?

JPG, PNG, GIF, TIFF, WebP, and BMP. If your format isn't listed, convert it to PNG first using any image viewer.

### Can I set the page size?

The default is one page per image at the image's native resolution. For a specific paper size (A4, Letter), you may need to resize images before combining — or use [Scale Pages](/tools/scale-pages) afterward.

### Is it safe for private photos?

Public tools see your images. Self-hosted Stirling-PDF processes them locally. For sensitive images, self-host.

### How do I make the combined PDF searchable?

Run [OCR PDF](/tools/ocr-pdf) on the output. OCR reads the text in the images and adds a searchable layer.

## Wrap-up

Combining images into a PDF is a drag-and-drop task. Open [Image to PDF](/tools/jpg-to-pdf), add your files, reorder, convert. Follow up with [Compress PDF](/tools/compress-pdf) if the result is too large, or [OCR PDF](/tools/ocr-pdf) if you need searchable text.
