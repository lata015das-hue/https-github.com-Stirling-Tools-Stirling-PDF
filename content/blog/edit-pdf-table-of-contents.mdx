# How to Edit a PDF Table of Contents (Bookmarks)

> **Short answer.** To edit a PDF's table of contents, open [Edit Table of Contents](/tools/edit-table-of-contents), upload the file, modify the bookmark structure, and download the updated version with corrected navigation.

PDF bookmarks are the clickable navigation panel in the sidebar of any PDF reader. When they're wrong, missing, or disorganized, readers can't navigate a long document efficiently. Editing fixes that without touching the actual content.

## What you'll need

- A PDF with missing or incorrect bookmarks.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Edit Table of Contents](/tools/edit-table-of-contents).
2. **Upload.** Drop the PDF.
3. **Edit bookmarks.** Add new entries, remove outdated ones, fix page targets, reorganize hierarchy.
4. **Save.** Download the PDF with updated bookmarks.

## When to edit bookmarks

- **After merging.** [Merge PDFs](/tools/merge-pdf) combines files but doesn't unify bookmarks. Add new top-level entries for each merged section.
- **After splitting.** Bookmarks may reference pages that no longer exist.
- **Scanned documents.** Scans never have bookmarks. Add them manually for key sections.
- **Long reports.** A 100-page document without bookmarks is unusable. Add chapter-level entries.

## Frequently asked questions

### Is it free to edit PDF bookmarks?

Yes. [Edit Table of Contents](/tools/edit-table-of-contents) is free, no signup.

### Does editing bookmarks change page content?

No. Bookmarks are metadata — navigation pointers. Page content is unchanged.

### Can I create bookmarks from scratch?

Yes. Upload a PDF with no bookmarks and add them manually.

### Is my file safe?

Public tools see your file. Self-hosted Stirling-PDF keeps it local.

### Do bookmarks survive compression?

Yes. [Compress PDF](/tools/compress-pdf) doesn't affect bookmarks.

## Wrap-up

PDF bookmarks make long documents navigable. Open [Edit Table of Contents](/tools/edit-table-of-contents), add or fix entries, download. Essential after [Merge PDFs](/tools/merge-pdf) or when working with scanned documents.
