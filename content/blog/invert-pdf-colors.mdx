# How to Invert or Replace Colors in a PDF

> **Short answer.** To invert colors in a PDF, open [Replace/Invert Colors](/tools/replace-invert-pdf), upload the file, and download the color-inverted version. White becomes black, black becomes white, and all other colors shift accordingly.

Dark mode for documents. A white-background PDF blasts your eyes at 2 AM. Inverting gives you a dark-background version that's easier on the eyes — without changing the content.

## What you'll need

- A PDF you want color-inverted.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Replace/Invert Colors](/tools/replace-invert-pdf).
2. **Upload.** Drop the PDF.
3. **Invert.** Click the action button.
4. **Download.** The output has inverted colors.

## Use cases

- **Dark mode reading.** Invert white-background documents for nighttime reading.
- **Accessibility.** Some readers with low vision prefer high-contrast inverted colors.
- **Pre-OCR enhancement.** For very dark scans, inverting before [OCR PDF](/tools/ocr-pdf) can improve text recognition accuracy.
- **Creative effects.** Negative-image style for artistic purposes.

## Common pitfalls

- **Photos look wrong.** Color inversion affects ALL elements including photos. The document may look unusual if it contains images.
- **Already dark background.** Inverting a dark-background PDF makes it white. Check first.
- **Print-unfriendly.** An inverted (dark background) PDF uses massive amounts of ink. Don't print it — it's for screen reading.

## Frequently asked questions

### Is it free to invert PDF colors?

Yes. [Replace/Invert Colors](/tools/replace-invert-pdf) is free, no signup.

### Does it affect text readability?

For text-only documents, inverted text (white on black) is perfectly readable. For documents with photos, results vary.

### Can I replace specific colors instead of inverting all?

Depends on the tool options. Basic inversion swaps all colors. Custom color replacement may require a PDF editor.

### Is my file safe?

Public tools see your file. Self-hosted Stirling-PDF keeps it local.

### Can I undo the inversion?

Run inversion again — inverting twice returns to the original colors.

## Wrap-up

Inverting PDF colors creates a dark-mode version of any document. Open [Replace/Invert Colors](/tools/replace-invert-pdf), upload, download. For pre-OCR enhancement on dark scans, invert before running [OCR PDF](/tools/ocr-pdf).
