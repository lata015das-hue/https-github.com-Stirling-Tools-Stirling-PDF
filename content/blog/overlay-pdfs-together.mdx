# How to Overlay One PDF on Top of Another

> **Short answer.** To overlay one PDF on top of another, open [Overlay PDFs](/tools/overlay-pdf), upload both files (the base and the overlay), and download the combined result where the overlay sits on top of the base, page by page.

Overlaying is different from merging. Merging puts pages in sequence (page 1, then page 2). Overlaying puts one page ON TOP of another — like placing a transparent letterhead over your document content.

## What you'll need

- Two PDF files: the base document and the overlay (letterhead, background, logo sheet).
- The overlay should have transparency where you want the base to show through.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Prepare the overlay.** Create a PDF with your letterhead, logo, or background. Make unused areas transparent.
2. **Open the tool.** Go to [Overlay PDFs](/tools/overlay-pdf).
3. **Upload both files.** Base document first, overlay second (or as directed by the tool).
4. **Overlay.** Click the action button.
5. **Download.** Each page of the base now has the overlay on top.

## Use cases

- **Company letterhead.** Overlay a letterhead template onto plain documents.
- **Watermark images.** Overlay a transparent PNG-based PDF as a visual watermark (when text watermarks from [Add Watermark](/tools/add-watermark-pdf) aren't enough).
- **Backgrounds.** Add a colored or textured background behind existing content.
- **Stamps.** Overlay an "APPROVED" stamp image positioned in the corner.

## Common pitfalls

- **Overlay covers content.** The overlay must have transparency. A solid white overlay hides everything underneath.
- **Page count mismatch.** If the overlay has 1 page and the base has 10, the overlay is typically repeated on all pages. Check the tool's behavior.
- **Wrong layer order.** Base goes behind, overlay goes on top. Reverse them if content is hidden.

## Frequently asked questions

### Is it free to overlay PDFs?

Yes. [Overlay PDFs](/tools/overlay-pdf) is free, no signup.

### What's the difference between overlay and merge?

Merge = pages in sequence (1, 2, 3...). Overlay = pages stacked on top of each other (layer by layer).

### Can I overlay a PNG image instead of a PDF?

Convert the PNG to a PDF first using [Image to PDF](/tools/jpg-to-pdf), then overlay.

### Is my file safe?

Public tools see your files. Self-hosted Stirling-PDF keeps them local.

### Can I remove an overlay after applying?

No. Overlaying is permanent. Keep the original base document.

## Wrap-up

Overlaying PDFs combines two documents layer-by-layer. Open [Overlay PDFs](/tools/overlay-pdf), upload base + overlay, download the combined result. For text watermarks, use [Add Watermark](/tools/add-watermark-pdf). For page merging (sequential), use [Merge PDFs](/tools/merge-pdf).
