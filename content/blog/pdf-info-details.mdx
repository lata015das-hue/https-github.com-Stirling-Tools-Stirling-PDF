# How to Get Detailed Information About a PDF File

> **Short answer.** To get detailed information about a PDF, open [PDF Info](/tools/get-info-on-pdf), upload the file, and view: page count, page dimensions, file size, metadata (author, title, dates), fonts, encryption status, and PDF version.

Before editing, compressing, or converting a PDF, knowing what's inside helps you choose the right approach. Is it encrypted? How many fonts? What version? Is it a scan or text-based? PDF Info answers all of this in one look.

## What you'll need

- A PDF you want to inspect.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [PDF Info](/tools/get-info-on-pdf).
2. **Upload.** Drop the PDF.
3. **View details.** The tool displays comprehensive information.

## What you'll learn

- **Page count and dimensions** — helps plan [Split PDF](/tools/split-pdf) or [Scale Pages](/tools/scale-pages) operations.
- **File size breakdown** — identifies why the file is large (images? fonts?).
- **Metadata** — title, author, creator, creation/modification dates.
- **Fonts embedded** — how many and which ones (helps diagnose display issues).
- **Encryption status** — whether a password is needed for [Remove Password](/tools/unlock-pdf).
- **PDF version** — compatibility with older readers.
- **JavaScript presence** — security concern, pair with [Show JavaScript](/tools/show-javascript).

## Frequently asked questions

### Is it free to check PDF info?

Yes. [PDF Info](/tools/get-info-on-pdf) is free, no signup.

### Does it modify the file?

No. Inspection is read-only. The file is unchanged.

### Can I see if a PDF is a scan or text-based?

Indirectly. If fonts are listed, it's likely text-based. If no fonts but large images per page, it's likely a scan.

### Is my file safe?

Public tools see your file. Self-hosted Stirling-PDF keeps it local.

### What if it shows encrypted?

Use [Remove Password](/tools/unlock-pdf) if you have the password, then re-inspect.

## Wrap-up

PDF Info gives you a complete diagnostic of any PDF before you work with it. Open [PDF Info](/tools/get-info-on-pdf), upload, read the details. Use the information to choose the right tools: [Compress PDF](/tools/compress-pdf) for large files, [OCR PDF](/tools/ocr-pdf) for scans, [Remove Password](/tools/unlock-pdf) for encrypted files.
