# How to Add a Watermark to a PDF Online Free

> **Short answer.** To add a watermark to a PDF online free, open [Add Watermark](/tools/add-watermark-pdf), type your watermark text (e.g., "DRAFT" or "CONFIDENTIAL"), upload the file, and download the watermarked copy.

Watermarks signal intent: "this is a draft," "this is confidential," "this belongs to us." Adding one after the fact — without going back to the source document — is a common need for legal, HR, and design review workflows.

## What you'll need

- The PDF you want to watermark.
- The watermark text (e.g., "DRAFT", "CONFIDENTIAL", your company name).
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Add Watermark](/tools/add-watermark-pdf).
2. **Type your watermark.** Enter the text. Keep it short — 1-2 words work best visually.
3. **Upload.** Drop the PDF into the upload area.
4. **Run.** The watermark is applied to every page.
5. **Download.** Verify the output looks right.

To remove a watermark later, you'd need to go back to the un-watermarked original — watermarking is a one-way operation by design.

## Use cases

- **Legal drafts.** Mark documents as "DRAFT" so no one treats a work-in-progress as final.
- **Confidential HR docs.** Stamp "CONFIDENTIAL" before sharing with limited audiences.
- **Client proofs.** Add your company name so clients don't share paid work before payment.
- **Internal review.** "FOR INTERNAL USE ONLY" signals the document shouldn't leave the org.

## Common pitfalls

- **Text too long.** "This document is confidential and should not be shared" wraps awkwardly. Use one word: "CONFIDENTIAL."
- **Watermark covers content.** Most tools use a diagonal, semi-transparent overlay. If yours doesn't, adjust opacity or position.
- **Forgot to watermark before sharing.** Once shared, you can't retroactively watermark someone else's copy. Watermark before distribution.

## Frequently asked questions

### Is it free to add a watermark to a PDF?

Yes. [Add Watermark](/tools/add-watermark-pdf) is free, no signup, no limit on pages when self-hosted.

### Can I use an image watermark instead of text?

The basic tool supports text. For image watermarks (logos), use [Overlay PDFs](/tools/overlay-pdf) with a transparent PNG as the overlay.

### Does the watermark make the file larger?

Negligibly. Text watermarks add a few KB at most. If size matters, run [Compress PDF](/tools/compress-pdf) afterward.

### Is my file safe?

Public tools see your document. Self-hosted Stirling-PDF keeps it on your machine. For confidential files, self-host.

### Can I watermark only specific pages?

Not directly with this tool. Workaround: [Extract Pages](/tools/extract-pages) the pages you want watermarked, watermark them, then [Merge PDFs](/tools/merge-pdf) with the unwatermarked pages.
