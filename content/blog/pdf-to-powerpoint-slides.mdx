# How to Convert a PDF to PowerPoint Slides

> **Short answer.** To convert a PDF to PowerPoint slides, open [PDF to PowerPoint](/tools/pdf-to-presentation), upload your PDF, and download the PPTX. Each PDF page becomes one slide. Edit in PowerPoint, Google Slides, or Keynote.

You received a presentation as a PDF — maybe from a client, a conference, or a colleague on a different OS. You need to edit slide 7. Without conversion, you're stuck. With it, you get a working PPTX in seconds.

## What you'll need

- The PDF containing the slides.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for confidential presentations.

## Step-by-step

1. **Open the tool.** Go to [PDF to PowerPoint](/tools/pdf-to-presentation).
2. **Upload.** Drop the PDF.
3. **Convert.** Click the action button.
4. **Download.** Open the PPTX in your preferred editor.
5. **Edit.** Each page is now a separate slide. Text may be editable; images are placed on the slide background.

## What converts well (and what doesn't)

| Converts well | May need manual fixes |
|---|---|
| Text blocks on simple layouts | Layered graphics and shapes |
| Images (placed as slide background) | Editable vector diagrams |
| Bullet lists | Animations (PDF has no animation data) |
| Clean single-column slides | Complex charts (may rasterize) |

Expect the output to be a starting point, not a perfect replica. The simpler the original PDF layout, the cleaner the conversion.

## Common pitfalls

- **Scanned slides.** If the PDF is from a scanner (no selectable text), the output will be image-only slides. Run [OCR PDF](/tools/ocr-pdf) first for text.
- **Fonts look different.** The output uses fonts available on your machine. If the original used a custom font not installed on yours, PowerPoint substitutes.
- **Large file.** PDFs with high-res backgrounds produce large PPTX files. Run [Compress PDF](/tools/compress-pdf) on the source before converting to reduce output size.

## Frequently asked questions

### Is it free to convert a PDF to PowerPoint?

Yes. [PDF to PowerPoint](/tools/pdf-to-presentation) is free, no signup, no watermark.

### Can I edit the text after conversion?

If the original PDF has real text (not a scan), the text is editable in the PPTX. For scans, run [OCR PDF](/tools/ocr-pdf) first.

### Does it work with Google Slides?

Yes. Download the PPTX, then open it in Google Slides via File > Import. Google Slides reads PPTX natively.

### Is my presentation safe?

Public tools see your file. Self-hosted Stirling-PDF processes it locally. For confidential presentations, self-host.

### Can I convert back to PDF after editing?

Yes. Export from PowerPoint as PDF, or use [File to PDF](/tools/file-to-pdf).

## Wrap-up

Converting a PDF to PowerPoint gives you editable slides from any PDF source. Open [PDF to PowerPoint](/tools/pdf-to-presentation), upload, download the PPTX. For scanned content, run [OCR PDF](/tools/ocr-pdf) first so you get real text on the slides.
