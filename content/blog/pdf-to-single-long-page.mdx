# How to Convert a Multi-Page PDF to One Long Page

> **Short answer.** To convert a multi-page PDF into one long page, open [PDF to Single Page](/tools/pdf-to-single-page), upload the file, and download a version where all pages are stitched vertically into one continuous page.

Page breaks make sense for printing but not for digital reading, infographics, or web-style scrolling. Converting to a single long page eliminates breaks and creates one continuous document.

## What you'll need

- A multi-page PDF you want as one page.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [PDF to Single Page](/tools/pdf-to-single-page).
2. **Upload.** Drop the multi-page PDF.
3. **Convert.** The tool stitches all pages vertically.
4. **Download.** One very tall page with all content in sequence.

## Use cases

- **Infographics.** Combine designed pages into one scrollable image.
- **Web display.** Embed as a single continuous view without page breaks.
- **Comparison.** See all pages at once without flipping.
- **Export to image.** Convert the single page to one tall JPG using [PDF to Image](/tools/pdf-to-jpg).

## Common pitfalls

- **Extremely long output.** A 50-page document becomes one very tall page. Some PDF readers struggle with pages over 200 inches.
- **Headers/footers repeat.** Each original page's header and footer are now in the middle of the long page. Crop them first with [Crop PDF](/tools/crop) if they're distracting.
- **Large file size.** A long page with many images can be very large. Run [Compress PDF](/tools/compress-pdf) afterward.

## Frequently asked questions

### Is it free?

Yes. [PDF to Single Page](/tools/pdf-to-single-page) is free, no signup.

### Can I convert back to multiple pages?

Not easily. The original page boundaries are gone. Keep the original multi-page version.

### Does it work with landscape pages?

Yes, but mixed orientations may produce odd results. Standardize orientation with [Rotate PDF](/tools/rotate-pdf) first.

### Is my file safe?

Public tools see your file. Self-hosted Stirling-PDF keeps it local.

### Can I choose horizontal stitching instead?

This tool stitches vertically (top to bottom). For side-by-side, use [Multi-Page Layout](/tools/multi-page-layout) with 2 pages per sheet.

## Wrap-up

Converting a multi-page PDF to one long page removes all page breaks. Open [PDF to Single Page](/tools/pdf-to-single-page), upload, download the continuous version. For the reverse (splitting), use [Split PDF](/tools/split-pdf).
