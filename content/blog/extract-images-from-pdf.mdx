# How to Extract Images from a PDF

> **Short answer.** To extract images from a PDF, open [Extract Images](/tools/extract-images), upload the file, and download all embedded images as separate files (JPG, PNG, etc.) at their original resolution.

A brochure has product photos you need for a presentation. A research paper has charts you want to reuse. Instead of screenshotting (which loses quality), extraction pulls out the original embedded image files.

## What you'll need

- A PDF containing images you want to extract.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Extract Images](/tools/extract-images).
2. **Upload.** Drop the PDF.
3. **Extract.** The tool finds every embedded image in the file.
4. **Download.** Each image downloads as a separate file at its original resolution.

## What gets extracted

- Photographs embedded in the document.
- Logos and graphics.
- Charts and diagrams (if stored as raster images).
- Scanned page images (if the PDF is a scan).

What does NOT get extracted: text rendered as vector paths, SVG graphics embedded as PDF drawing commands, or backgrounds generated by CSS-like styling.

## Common pitfalls

- **No images found.** The "images" you see might be vector graphics (drawn by the PDF renderer, not embedded as image files). Nothing to extract.
- **Low resolution output.** The images are only as good as what was embedded. If someone pasted a 72-DPI screenshot, that's what you get.
- **Too many images.** A 100-page scanned document produces 100 images (one per page). That's correct — each scan page IS an image.

## Frequently asked questions

### Is it free to extract images from a PDF?

Yes. [Extract Images](/tools/extract-images) is free, no signup.

### What format are the extracted images?

Usually JPEG or PNG, depending on how they were embedded in the PDF. The tool preserves the original format.

### Does it work on scanned PDFs?

Yes — each scanned page is itself an image. You'll get one image per page.

### Is my file safe?

Public tools see your PDF. Self-hosted Stirling-PDF keeps it local.

### Can I extract images from only certain pages?

[Extract Pages](/tools/extract-pages) the pages you want first, then run image extraction on the result.

Extracting images from a PDF gives you the original embedded files at their full resolution. Open [Extract Images](/tools/extract-images), upload, download. For page-specific extraction, combine with [Extract Pages](/tools/extract-pages).
