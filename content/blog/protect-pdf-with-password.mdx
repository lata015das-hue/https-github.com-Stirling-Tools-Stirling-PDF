# How to Protect a PDF with a Password

> **Short answer.** To protect a PDF with a password, open [Add Password](/tools/add-password), type a strong password, upload the file, and download the encrypted copy.

Sending a contract via email? A password-protected PDF ensures only the intended recipient can read the file even if intercepted.

## What you'll need

- The PDF you want to protect.
- A strong password (12+ characters, mix of letters/numbers/symbols).
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Add Password](/tools/add-password).
2. **Type your password.** Make it strong. Do not reuse passwords.
3. **Upload.** Drop the PDF.
4. **Encrypt.** Click the action button.
5. **Download.** The encrypted PDF requires the password to open.
6. **Share the password separately.** Never put it in the same email as the file.

## User password vs. owner password

- **User password**: required to open the PDF at all.
- **Owner password**: controls actions (print, copy, edit) but allows viewing. Use [Change Permissions](/tools/change-permissions) for this.

## Common pitfalls

- **Weak password.** Use a passphrase: "correct-horse-battery-staple" beats "P@ssw0rd".
- **Password sent with the file.** Send via different channels (file by email, password by SMS).
- **Lost password.** No recovery exists. Keep the original unencrypted file backed up.

## Frequently asked questions

### Is it free to password-protect a PDF?

Yes. [Add Password](/tools/add-password) is free, no signup, no watermark.

### How strong is the encryption?

AES-256 encryption, the same standard used by banks and governments.

### Can someone crack the password?

With a weak password, yes. With a strong 16+ character passphrase, effectively no.

### Is my file safe during the process?

Public tools see your unencrypted file. Self-hosted Stirling-PDF keeps it on your machine. For sensitive documents, self-host.

### How do I remove the password later?

Use [Remove Password](/tools/unlock-pdf) if you know the current password.

## Wrap-up

Protecting a PDF with a password takes 30 seconds. Open [Add Password](/tools/add-password), set a strong password, download the encrypted file. To remove later: [Remove Password](/tools/unlock-pdf). To restrict actions without blocking access: [Change Permissions](/tools/change-permissions).
