# How to Add Attachments to a PDF

> **Short answer.** To add attachments to a PDF, open [Add Attachments](/tools/add-attachments), upload the PDF and the files you want to embed, and download the PDF with the files bundled inside.

A PDF can carry other files inside it — like an email attachment but embedded in the document itself. The reader opens the PDF and can extract the embedded files. Useful for bundling source data with a report, or delivering a complete package as one file.

## What you'll need

- A PDF you want to add files to.
- The files to embed (spreadsheets, images, text files, other PDFs).
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Add Attachments](/tools/add-attachments).
2. **Upload the PDF.** This is the container.
3. **Upload attachments.** Select the files to embed.
4. **Attach.** Click the action button.
5. **Download.** The PDF now contains the embedded files. Recipients can extract them from the PDF reader's attachments panel.

## Use cases

- **Reports with source data.** Embed the Excel spreadsheet alongside the PDF charts.
- **Contracts with exhibits.** Bundle exhibits as attachments rather than separate emails.
- **Academic papers with datasets.** Attach raw data for reproducibility.
- **Self-contained packages.** One file to share, nothing to lose.

## Common pitfalls

- **File size grows.** Each attachment adds its full size to the PDF. A 5 MB spreadsheet makes the PDF 5 MB larger.
- **Not all readers show attachments.** Basic PDF viewers may not display the attachments panel. Adobe Reader and most desktop readers do.
- **Security scanners may block.** Some email systems strip PDFs with embedded executables. Don't attach .exe files.

## Frequently asked questions

### Is it free to add attachments?

Yes. [Add Attachments](/tools/add-attachments) is free, no signup.

### What file types can I attach?

Any file type. Common: .xlsx, .csv, .docx, .png, .zip. Avoid executables (.exe) as they may be blocked by email scanners.

### Can recipients extract the attachments?

Yes. In Adobe Reader: View → Show/Hide → Navigation Panes → Attachments. Other readers have similar options.

### Is my file safe?

Public tools see your files. Self-hosted Stirling-PDF keeps them local.

### Does it change the visible pages?

No. Attachments are metadata — invisible unless the reader opens the attachments panel. Page content is unchanged.

## Wrap-up

Adding attachments to a PDF creates self-contained packages. Open [Add Attachments](/tools/add-attachments), upload PDF + files, download the bundled result. For security inspection of received attachments, use [Sanitize PDF](/tools/sanitize-pdf).
