# How to Unlock Locked PDF Form Fields

> **Short answer.** To unlock locked PDF form fields, open [Unlock PDF Forms](/tools/unlock-pdf-forms), upload the form, and download the unlocked version with editable fields restored.

Government forms, application PDFs, and contract templates sometimes lock fields after filling — preventing corrections. If you have the right to edit the form (it's yours, or you're the issuer), unlocking restores editability.

## What you'll need

- A PDF form with locked/read-only fields.
- A browser.
- Optional: self-hosted [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) for privacy.

## Step-by-step

1. **Open the tool.** Go to [Unlock PDF Forms](/tools/unlock-pdf-forms).
2. **Upload.** Drop the locked form.
3. **Unlock.** Click the action button.
4. **Download.** Fields are now editable again.
5. **Edit.** Fill or correct values in any PDF reader.

## Locked fields vs. password protection

- **Locked fields:** the form works but specific fields are read-only. Unlocking makes them editable.
- **Password protection:** the entire PDF is encrypted. Use [Remove Password](/tools/unlock-pdf) instead.

## Common pitfalls

- **Form was flattened, not locked.** If someone ran [Flatten PDF](/tools/flatten), the fields are gone permanently (burned into the page). Unlocking won't help — you need the original un-flattened version.
- **Legal implications.** Only unlock forms you have the right to modify. Don't alter submitted legal documents without authority.

## Frequently asked questions

### Is it free to unlock PDF form fields?

Yes. [Unlock PDF Forms](/tools/unlock-pdf-forms) is free, no signup.

### Will it unlock password-protected PDFs?

No. For password protection, use [Remove Password](/tools/unlock-pdf). This tool specifically handles locked form fields.

### Does it preserve existing filled data?

Yes. The field values stay; only the lock is removed.

### Is my file safe?

Public tools see your form data. Self-hosted Stirling-PDF keeps it local. For forms with personal information, self-host.

### Can I re-lock fields after editing?

Not directly with this tool. You'd need to re-create the form with locked properties in a PDF editor.

## Wrap-up

Unlocking PDF form fields restores editability to read-only forms. Open [Unlock PDF Forms](/tools/unlock-pdf-forms), upload, edit freely. For password-encrypted files, use [Remove Password](/tools/unlock-pdf) instead.
